# Leetcode 961 重复N次的元素
***
### 题目描述
在大小为 `2N` 的数组 `A` 中有 `N+1` 个不同的元素，其中有一个元素重复了 `N` 次。

返回重复了 `N` 次的那个元素。

**示例1:**  

	输入：[1,2,3,3]
	输出：3

**示例2:**  

	输入：[2,1,2,5,3,2]
	输出：2
	
**示例3:**  

	输入：[5,1,5,2,5,3,5,4]
	输出：5

**注意：**

* `4 <= A.length <= 10000`
* `0 <= A[i] < 10000`
* `A.length` 为偶数


### 考点

哈希表


### 代码
执行用时: **48ms**, 内存消耗: **14MB**

```
class Solution:
    def repeatedNTimes(self, A: List[int]) -> int:
        S = set()
        for a in A:
            if a in S:
                return a
            S.add(a)
```
