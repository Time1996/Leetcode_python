# Leetcode 面试题 17 打印从1到最大的n位数
***
### 题目描述

输入数字 n，按顺序打印出从 1 到最大的 n 位十进制数。比如输入 3，则打印出 1、2、3 一直到最大的 3 位数 999。

**示例1：**    

	输入: n = 1
	输出: [1,2,3,4,5,6,7,8,9]
	
	
**说明：**

* 用返回一个整数列表来代替打印
* n 为正整数


**考点**

数学


### 代码
执行用时: **48ms**, 内存消耗: **19.5MB**

```
class Solution:
    def printNumbers(self, n: int) -> List[int]:
        return [i for i in range(1, 10**n)]
```
