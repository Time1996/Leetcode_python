# Leetcode 136 只出现一次的数字
***
### 题目描述
给定一个**非空**整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。  

**说明：**   
  
  你的算法应该具有线性时间复杂度。你可以不使用额外空间来实现吗？


**示例1:**   
	
	输入: [2,2,1]
	输出: 1
	
**示例2:**   
	
	输入: [4,1,2,1,2]
	输出: 4
	

### 考点

* 位操作


### 代码  
执行用时: **60ms**, 内存消耗: **14.4MB**

```
class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        theone = 0
        for x in nums:
            theone ^= x
        return theone                       
```
